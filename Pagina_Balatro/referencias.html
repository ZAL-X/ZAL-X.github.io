Bootstrap es un framework de desarrollo web gratuito y de código abierto. Está diseñado para facilitar el proceso de desarrollo de los sitios web responsivos y orientados a los dispositivos móviles, proporcionando una colección de sintaxis para diseños de plantillas. https://www.hostinger.com/mx/tutoriales/que-es-bootstrap

Font Awesome es un framework de iconos vectoriales y estilos css. Pero… ¿para qué sirve? Este framework es utilizado para sustituir imágenes de iconos comunes por gráficos vectoriales convertidos en fuentes. Para ello utiliza una librería de mas de 400 iconos transformadas en fuentes. https://www.aquihaydominios.com/blog/font-awesom-que-es-y-como-se-usa/

Phoenix Code es un editor de código online gratuito que permite programar directamente desde el navegador sin instalaciones. https://proyectodescartes.org/escenas-aux/CurspLibrosDescartesHerramientasIA/recursos1/manual_phoenix.html

GitHub es una herramienta esencial para los ingenieros de software que ayuda a mejorar el flujo de trabajo y promueve la colaboración. 94 millones de desarrolladores, 4 millones de organizaciones y 330 millones de repositorios forman parte de la plataforma, y la comunidad de GitHub sigue creciendo. https://ebac.mx/blog/que-es-github

Google Fonts es un servicio de distribución de fuentes tipográficas propiedad de Google. Incluye una amplia biblioteca de fuentes de código abierto, un directorio web interactivo para navegar por la biblioteca y una API para usar las fuentes a través de CSS y Android. https://es.wikipedia.org/wiki/Google_Fonts

Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código. https://es.wikipedia.org/wiki/Visual_Studio_Code

Proyecto Balatro es un pequeño proyecto hecho por mi, un alumno de la UPSLP, donde recopilo información del videojuego y la hago visible en esta pagina web :p

Wikipedia es una enciclopedia libre, políglota y editada de manera colaborativa. Es administrada por la Fundación Wikimedia, una organización sin ánimo de lucro cuya financiación está basada en donaciones. https://es.wikipedia.org/wiki/Wikipedia

Steam es una plataforma de distribución digital de videojuegos desarrollada por Valve Corporation. Fue lanzada el 12 de septiembre de 2003 como una forma para Valve de proveer actualizaciones automáticas a sus juegos, pero finalmente se amplió para incluir juegos de terceros. https://es.wikipedia.org/wiki/Steam

Xbox es una marca de videojuegos creada por y propiedad de Microsoft que incluye una serie de video-consolas desarrolladas por la misma compañía, de sexta a novena generación, así como aplicaciones (juegos), servicios de streaming y el servicio en línea Xbox Live. La marca fue introducida por primera vez el 8 de noviembre de 2001, en los Estados Unidos, con el lanzamiento de la consola Xbox. https://es.wikipedia.org/wiki/Xbox_(marca)
